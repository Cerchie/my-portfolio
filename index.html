My Projects

Mars Photo Album

This project raises awareness of NASA's presence on Mars. It educates the user about the data we have available to us and about the technology used to acquire the data.

It pulls from two APIs, NASA's Mars Rover Photos Viewer for the Mars photos, and from NASA's Mars Weather Service API for the weather widget. I used Python, Flask, Axios, SQLAlchemy, CSS, and HTML to create the features I needed for this project.

I used Flask to write the routes, Axios to make my API calls, SQLAlchemy to build my database, CSS Bootstrap to style my website, and Jinja to pull variables from my routes into my HTML templates.

NASA SpaceApps 2020

This was a collaborative effort in response to NASA's Confront-Better Together challenge for the annual hackathon. My team and I built an app designed to help people practice responding to biased comments.

We used Python, Flask, and Jinja templating to create our website, with some SQALchemy on the backend. My team and I enjoyed working through problems together from four different states, across 3 different time zones.

At the end of our time, we were proud to submit a minimum viable product to NASA. We also learned how to use remote communication effectively.
